What it is

Automatic tool for financial software quality assessment

Arcan is a tool for software developers and tech product managers that automatically estimates the cost of keeping and solving Technical Debt (code inefficiencies). With both metrics, you can decide if and when to invest in improving your system's quality.

Live demo

How it works

Step 1: input

Arcan automatically scans the source code of your project from your local folder or from your online repository.

Step 2: analysis

Arcan reverse engineers your code and runs a set of automatic analysis for quality assessment and architectural anomalies detection.

Step 3: knowledge

The Arcan analysis outputs the architectural map of your project and the results of the quality assessment.
